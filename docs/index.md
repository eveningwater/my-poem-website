# 五言绝句格律

一二对，三四对，二三粘。

对是平仄反，粘是平仄同，韵脚一二声。

一三不论，二四分明。

五言绝句格律成！

# 七言绝句格律

一二对，三四对，二三粘，

对是平仄反，粘是平仄同，韵脚一二声。

一三五不论，二四六分明，七言绝句格律成。

五言，七言，忌三仄尾，三平尾，孤平。两首五言构成五律，两首七言构成七律，只不过五律和七律中间两联需要对仗。
