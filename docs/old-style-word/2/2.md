# 诉衷情

语动流水心甚乱，倍思闲。

尝望天，复见，日悠然。

争忍不为情？又羞赧。

送我心，安你心，已然迷伊颜。

## 自译

你的一言一行触动了我，让我的心变得无比的慌乱，更加的想恋和你在一起空闲的玩耍时间。

经常抬头望着天空，每天都是在忧愁着再次相见的日子。

怎么可能会忍着不动情呢？可是啊我又害羞着不敢向你诉说这段情意。

只想把我的心送给你，安抚你的心，原来不知不觉中我已经迷上了你的一切。
