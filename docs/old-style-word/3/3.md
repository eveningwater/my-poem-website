# 踏莎行

不解天公,几番风雨,而今终得日开去。

落光似火炙人生,莫如星月只点煜。

我道天公,应语盘路,此行便是将来苦。

红尘万事有来回,世物一命皆无住。

## 自译

不理解老天，来了几次风雨，如今终于能够有太阳照射了。

可是那落下来多的阳光就好像火一样炙烤着世间，还不如星星月亮的那点点光芒。

我想说老天，应该是想对我说这是一条蜿蜒曲折的道路，这是一番对未来的辛苦的磨练。

这个世间万事万物都会有循环的来往，生命也只有一次随着时间的流逝永远不会停止。
