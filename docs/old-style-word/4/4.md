# 卜算子-咏地铁

才有空闲处，又入几多愁。

向来难知如虎踞，身心岂可游？

人皆飞怨语，此情何时休？

待到隔开不宜去，南北竟分头。

## 自译

才腾出来空闲的地方，结果又进来不少人。

一向知道自己很难像老虎一样盘踞，身心都没办法移动了。

人们都不停的抱怨着，这种情况到底什么时候可以结束？

等到站了可以离去之时，却不适合离去，因为竟然搞错了上下车的两个方向。
